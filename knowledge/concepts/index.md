# Domain Entity

* [Concept Document](/concepts/concept-document.md) - One UTF-8 Markdown knowledge unit with YAML frontmatter and a structured body.
* [Knowledge Bundle](/concepts/knowledge-bundle.md) - A self-contained directory tree of OKF Markdown documents used as the unit of distribution and retrieval.
* [Knowledge Graph](/concepts/knowledge-graph.md) - Directed relationships extracted from standard Markdown links between concept documents.
* [Noli](/concepts/noli.md) - Your friendly knowledge-format builder.
* [Prepared Agent Context](/concepts/prepared-agent-context.md) - A prebuilt bounded Markdown context plus a checksum manifest for an agent task.
* [Project Configuration](/concepts/project-configuration.md) - The strict repository-local noli.yaml policy used for generation, validation, retrieval defaults, and paths.
* [Structured Concept Source](/concepts/structured-concept-source.md) - The deterministic YAML input from which generated concept Markdown and graph links are rendered.
