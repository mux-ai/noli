# Nimbus Hub X100 support manual

> Demonstration data for Noli. Nimbus Hub X100 is a fictional product.

## Product overview

Nimbus Hub X100 is a demonstration network hub used in this example. Its model identifier is X100.

## Wi-Fi connection failure

If the status screen says **Cannot connect to Wi-Fi**, first verify that the saved network name and password match the router settings. Restart the hub after correcting the credentials. The connection is restored when the status screen shows **Online**.

This troubleshooting procedure applies to Nimbus Hub X100.

## Error E102

Error code E102 means that the demonstration hub could not authenticate with the configured Wi-Fi network. Incorrect credentials are a documented cause. Resolve E102 by verifying the saved credentials and restarting the hub.
